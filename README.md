## 💡 Using The Interface

The main workspace holds your tools and layers. You will see a toolbar on the left side of the screen. This bar contains your brushes, erasers, and shape tools. The layer window sits on the right side. Use this to maintain order in your artwork. Create new layers for sketches, line art, and colors. 

To use a 3D model, look for the "Material" panel. Drag and drop a model onto your canvas. Use the object tool to rotate and pose the model. You can change the lighting and camera angle to fit your scene.

If you create a comic, use the "Story" menu to add pages. You can view your whole comic in the "Page Manager." This tool helps you keep track of your storyboard and chapter flow.

## 🛠 Troubleshooting

Most installation issues have simple fixes.

- **Installer does not open:** Check your antivirus settings. Sometimes security software blocks new files. Disable it for a moment, then try the install again.
- **Software runs slowly:** Close other heavy programs like web browsers. Large brushes or many layers can take a lot of memory.
- **Tablet pressure not working:** Go to "File" and then "Preferences." Choose the "Tablet" tab. Ensure the settings match the brand of your tablet.
- **Update failure:** Visit the release page again. Download the newest version of the installer and run it over the old version.
